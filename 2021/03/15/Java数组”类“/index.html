Java数组“类”

​ Java数组的用法与C,CPP中数组方法类似，其基本作用时存储一个元素个数固定且元素类型相同的有序集，但Java数组中包含了一部分面向对象的特性。

数组声明

数组名只是一个引用变量，如果变量不包含对数组的引用，则这个引用的值为NULL，且数组中的元素也是引用

创建方式如下

有以下几个注意点

  • 数组在被分配空间后，数组大小便再也不可改变，可以使用arrayRefVar.length来获取数组的长度
  • 创建数组后，其元素被赋予默认值
  • 数组中的元素都是引用

数组支持数组初始化语法elementType[] arrayRefVar= {value0,value1...valuek},但使用时必须将声明，创建和初始化数组都放到一条语句中。

tip:可以使用一条语句打印char型数组

数组的复制

假设已经有两个double数组list1,list2

使用list1=list2的结果是使list1,list2指向同一个引用。

真正复制方法有以下几种

  1. 使用循环
  2. 使用System中的static方法arraycopy
  3. 使用clone方法复制数组

数组其他特性

在Java中如果访问超过边界的数组，就会直接抛出一个异常，以避免访问到不该访问的位置。

数组的本质

  1. 数组也是类,虚拟机运行的时候创建
  2. 数组类不包括任何成员和变量
  3. 数组的lehgth是final的，因此无法改变，也无法扩容
  4. 不同类型的数组，其类型亦不同，例如，int[]的类名为[I,int[][]的类名为[[I
